## Selected Project Themes

### FieldViewer

FieldViewer is a digital field-visualization concept designed to support integrated field review and technical decision-making. It brings together maps, wells, production data, timelines, completion information, well-test results, ESP performance reviews, real-time field data streams such as production and drilling data, and other technical outputs within a web-based environment that is accessible, interactive, and practical for technical decision-makers.

The objective is to provide a more interactive, reusable, and data-connected alternative to static PowerPoint-based field reviews and disconnected technical datasets.


### Seismic Attribute Machine Learning

I have developed workflows for seismic-attribute-driven reservoir property prediction using Python, well-calibrated petrophysical data, regression methods, clustering concepts, and blind-well validation.

These workflows focus on geological consistency, QC, calibration, and interpretability rather than black-box prediction.

### STOIIP estimation and Reservoir Uncertainty

I have developed and extended Streamlit-based workflows for STOIIP estimation using Monte Carlo simulation, uncertainty handling, ZMAP reservoir grids, spatial mapping, and 3D visualization.

### Fault and Amplitude Profile Analysis

I have built Python workflows for extracting fault-normal profiles, analyzing seismic or grid values around interpreted fault segments, and supporting structural interpretation QC through repeatable visual outputs.

### Geoscience Data Utilities

I have developed utilities for ZMAP grids, GeoTIFF handling, data format conversion between different geoscience platforms ( Petrel/DSG, etc.), Excel-to-LAS conversion, report building, image-to-presentation workflows, and other repetitive technical tasks commonly encountered in geoscience and reservoir studies.
