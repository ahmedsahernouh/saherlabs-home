FieldViewer

Integrated field review for maps, wells, production, and AI-assisted analysis

FieldViewer is a web-based oilfield visualization concept for combining maps, wells, production, timelines, completions, well testing, database review, and AI-assisted technical context in one browser workflow.

Open FieldViewer AI Lab concept Screenshot tour

What It Addresses

Oilfield reviews often depend on disconnected maps, spreadsheets, presentations, completion files, production charts, and technical notes. FieldViewer demonstrates a more reusable review environment where related field context can be explored from a single web interface.

  • AudienceSubsurface, reservoir, production, and digital transformation teams.
  • StackPython, Bokeh, web maps, structured field data, SQLite concepts, and AI-assisted workflows.
  • Live appfieldviewer.saherlabs.dev

Main Modules

Key areas covered by the FieldViewer field-review workflow.

Maps and wells

Field map context, well locations, selected-well summaries, and surface/subsurface layers.

Production and timelines

Production history, bubble timelines, well profiles, and rate review screens.

Status dashboards

Daily operating status views for wells, rigs, field totals, and operational context.

Completions and testing

Completion image review, well testing trends, crossplots, and map-linked screening.

Database review

Structured data access concepts for technical tables and field metadata.

AI Lab

Assistant-style interaction with selected wells, field context, and controlled database-style questions.

Technical Concept

FieldViewer is designed as a practical field-review layer for organizing maps, wells, production, completion, testing, and AI-assisted context into browser modules that support faster review, repeatable communication, and more traceable technical screening.
